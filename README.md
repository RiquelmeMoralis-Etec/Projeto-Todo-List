Execute o seguinte comando antes de testar
lembre-se de estar na pasta FrontEnd no terminal antes de executar

```
npm install
```

Após isso execute
isso executara o frontend

```
npm run dev
```

Para executar o backend esteja na pasta BackEnd no terminal e execute a maquina virtual

```
.venv\Scripts\activate
```

Após fazer isso execute
isso ira executar seu backend

```
python.exe app.py
```

claro, você precisará ter coisa como pymongo,axios,flask_cors instalado em sua maquina tambêm.
